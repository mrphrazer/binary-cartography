# Binary Cartography

*Technical webinars on reverse engineering, malware analysis, and software protection.*

This repository hosts materials for **Binary Cartography**, Tim's independent webinar series on binary security. The focus is practical reverse engineering, malware analysis, and software protection, with materials for technical security professionals, reverse engineers, and analysts working on real binaries in practice.

Each webinar folder contains the session abstract together with slides, code, samples, references, and recording links where available.

## About Tim

Tim Blazytko is a binary security researcher and reverse-engineering expert with a PhD in program analysis. He focuses on independent consulting and hands-on trainings in reverse engineering, malware analysis, and software protection, and regularly contributes to the community through conference talks, research papers, and open-source tools.
